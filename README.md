# Xolvon Project Incubator Program

Landing page pendaftaran untuk Xolvon Project Incubator Program — tantangan inkubasi intensif 48 jam khusus Web Developer.

## Struktur

```
├── index.html       # Halaman utama
├── css/style.css    # Custom styles
├── js/script.js     # Interaktivitas (mobile menu, scroll)
└── README.md
```

## Tech Stack

- **Tailwind CSS** (CDN) — utility-first styling
- **Inter + Outfit** — Google Fonts
- **Lucide Icons** — icon set via CDN
- **Vanilla JS** — tanpa framework berat

## Cara Pakai

Buka `index.html` langsung di browser, atau deploy ke Vercel/Netlify.
